\mbox{\hyperlink{classFacet}{Facet}} used to register change filters, which are called for each transaction (unless explicitly \href{https://codemirror.net/6/docs/ref/\#state.TransactionSpec.filter}{\texttt{ disabled}}), and can suppress part of the transaction\textquotesingle{}s changes.
