\doxysubsection{Detailed Description}
\mbox{\hyperlink{classContext}{Context}} trackers are used to track stateful context (such as indentation in the Python grammar, or parent elements in the X\+ML grammar) needed by external tokenizers. 

You declare them in a grammar file as {\ttfamily @context export\+Name from \char`\"{}module\char`\"{}}.

\mbox{\hyperlink{classContext}{Context}} values should be immutable, and can be updated (replaced) on shift or reduce actions.

The export used in a {\ttfamily @context} declaration should be of this type.
