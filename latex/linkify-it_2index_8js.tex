new Linkify\+It(schemas, options)
\begin{DoxyItemize}
\item schemas (Object)\+: Optional. Additional schemas to validate (prefix/validator)
\item options (Object)\+: \{ fuzzy\+Link$\vert$fuzzy\+Email$\vert$fuzzy\+IP\+: true$\vert$false \}
\end{DoxyItemize}

Creates new linkifier instance with optional additional schemas. Can be called without {\ttfamily new} keyword for convenience.

By default understands\+:


\begin{DoxyItemize}
\item {\ttfamily http(s)\+://...} , {\ttfamily \href{ftp://..}{\texttt{ ftp\+://..}}.}, {\ttfamily mailto\+:...} \& {\ttfamily //...} links
\item \char`\"{}fuzzy\char`\"{} links and emails (example.\+com, \href{mailto:foo@bar.com}{\texttt{ foo@bar.\+com}}).
\end{DoxyItemize}

{\ttfamily schemas} is an object, where each key/value describes protocol/rule\+:


\begin{DoxyItemize}
\item {\bfseries{key}} -\/ link prefix (usually, protocol name with {\ttfamily \+:} at the end, {\ttfamily skype\+:} for example). {\ttfamily linkify-\/it} makes shure that prefix is not preceeded with alphanumeric char and symbols. Only whitespaces and punctuation allowed.
\item {\bfseries{value}} -\/ rule to check tail after link prefix
\begin{DoxyItemize}
\item {\itshape String} -\/ just alias to existing rule
\item {\itshape Object}
\begin{DoxyItemize}
\item {\itshape validate} -\/ validator function (should return matched length on success), or {\ttfamily Reg\+Exp}.
\item {\itshape normalize} -\/ optional function to normalize text \& url of matched result (for example, for @twitter mentions).
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}

{\ttfamily options}\+:


\begin{DoxyItemize}
\item {\bfseries{fuzzy\+Link}} -\/ recognige U\+R\+L-\/s without {\ttfamily http(s)\+:} prefix. Default {\ttfamily true}.
\item {\bfseries{fuzzy\+IP}} -\/ allow I\+Ps in fuzzy links above. Can conflict with some texts like version numbers. Default {\ttfamily false}.
\item {\bfseries{fuzzy\+Email}} -\/ recognize emails without {\ttfamily mailto\+:} prefix. 
\end{DoxyItemize}\mbox{\Hypertarget{linkify-it_2index_8js_ac2b9543e2b75679e660fc3f5d79eaf68}\label{linkify-it_2index_8js_ac2b9543e2b75679e660fc3f5d79eaf68}}
