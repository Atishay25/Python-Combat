\doxysubsection{Detailed Description}
A range set stores a collection of \href{https://codemirror.net/6/docs/ref/\#state.Range}{\texttt{ ranges}} in a way that makes them efficient to \href{https://codemirror.net/6/docs/ref/\#state.RangeSet.map}{\texttt{ map}} and \href{https://codemirror.net/6/docs/ref/\#state.RangeSet.update}{\texttt{ update}}. 

This is an immutable data structure.
