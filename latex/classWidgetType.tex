\doxysubsection{Detailed Description}
Widgets added to the content are described by subclasses of this class. 

Using a description object like that makes it possible to delay creating of the D\+OM structure for a widget until it is needed, and to avoid redrawing widgets even if the decorations that define them are recreated.
