Log an exception as an error.

Prior to J\+S\+Doc 3.\+3.\+0, this method would either log the exception (if lenient mode was enabled) or re-\/throw the exception (default).

In J\+S\+Doc 3.\+3.\+0 and later, lenient mode has been replaced with strict mode, which is disabled by default. If strict mode is enabled, calling the {\ttfamily handle} method causes J\+S\+Doc to exit immediately, just as if the exception had been re-\/thrown.
