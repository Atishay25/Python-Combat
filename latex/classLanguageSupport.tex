\doxysubsection{Detailed Description}
This class bundles a \href{https://codemirror.net/6/docs/ref/\#language.Language}{\texttt{ language}} with an optional set of supporting extensions. 

\mbox{\hyperlink{classLanguage}{Language}} packages are encouraged to export a function that optionally takes a configuration object and returns a {\ttfamily \mbox{\hyperlink{classLanguageSupport}{Language\+Support}}} instance, as the main way for client code to use the package.
