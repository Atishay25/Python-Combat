\item enable/disable rules
\item add/replace rules
\item allow assign rules to additional named chains (in the same)

\end{DoxyItemize}

You will not need use this class directly until write plugins. For simple rules control use \mbox{[}\mbox{[}Markdown\+It.\+disable\mbox{]}\mbox{]}, \mbox{[}\mbox{[}Markdown\+It.\+enable\mbox{]}\mbox{]} and \mbox{[}\mbox{[}Markdown\+It.\+use\mbox{]}\mbox{]}.
