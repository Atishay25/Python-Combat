js2xmlparser is a Node.\+js module that parses Java\+Script objects into X\+ML.\hypertarget{md_node_modules_js2xmlparser_README_autotoc_md1220}{}\doxysection{Features}\label{md_node_modules_js2xmlparser_README_autotoc_md1220}
Since X\+ML is a data-\/interchange format, js2xmlparser is designed primarily for J\+S\+O\+N-\/type objects, arrays and primitive data types, like many of the other Java\+Script to X\+ML parsers currently available for Node.\+js.

However, js2xmlparser is capable of parsing any object, including native Java\+Script objects such as {\ttfamily Date} and {\ttfamily Reg\+Exp}, by taking advantage of each object\textquotesingle{}s {\ttfamily to\+String} function or, if this function does not exist, the {\ttfamily String} constructor.

js2xmlparser also has support for the {\ttfamily Map} and {\ttfamily Set} objects introduced in E\+C\+M\+A\+Script 2015, treating them as J\+S\+O\+N-\/type objects and arrays respectively. Support for {\ttfamily Map}s is necessary to generate X\+ML with elements in a specific order, since J\+S\+O\+N-\/type objects do not guarantee insertion order. {\ttfamily Map} keys are always converted to strings using the method described above.

js2xmlparser also supports a number of constructs unique to X\+ML\+:


\begin{DoxyItemize}
\item attributes (through an attribute property in objects)
\item mixed content (through value properties in objects)
\item multiple elements with the same name (through arrays)
\end{DoxyItemize}

js2xmlparser can also pretty-\/print the X\+ML it outputs.\hypertarget{md_node_modules_js2xmlparser_README_autotoc_md1221}{}\doxysection{Installation}\label{md_node_modules_js2xmlparser_README_autotoc_md1221}
