\doxysubsection{Detailed Description}
\href{\#lr.ExternalTokenizer}{\texttt{ Tokenizers}} interact with the input through this interface. 

It presents the input as a stream of characters, tracking lookahead and hiding the complexity of \href{\#common.Parser.parse^ranges}{\texttt{ ranges}} from tokenizer code.

{\ttfamily .peek(0)} equals {\ttfamily .next}, {\ttfamily .peek(-\/1)} gives you the previous character, and so on.
