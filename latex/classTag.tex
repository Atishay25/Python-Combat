\doxysubsection{Detailed Description}
Highlighting tags are markers that denote a highlighting category. 

Represents a single doclet tag.

They are \href{\#highlight.styleTags}{\texttt{ associated}} with parts of a syntax tree by a language mode, and then mapped to an actual C\+SS style by a \href{\#highlight.Highlighter}{\texttt{ highlighter}}.

Because syntax tree node types and highlight styles have to be able to talk the same language, Code\+Mirror uses a mostly {\itshape closed} \href{\#highlight.tags}{\texttt{ vocabulary}} of syntax tags (as opposed to traditional open string-\/based systems, which make it hard for highlighting themes to cover all the tokens produced by the various languages).

It {\itshape is} possible to \href{\#highlight.Tag^define}{\texttt{ define}} your own highlighting tags for system-\/internal use (where you control both the language package and the highlighter), but such tags will not be picked up by regular highlighters (though you can derive them from standard tags to allow highlighters to fall back to those).

The text following the tag (for example, {\ttfamily text} in {\ttfamily @title text}).

Whitespace is trimmed from the tag text as follows\+:


\begin{DoxyItemize}
\item If the tag\textquotesingle{}s {\ttfamily keeps\+Whitespace} option is falsy, all leading and trailing whitespace are removed.
\item If the tag\textquotesingle{}s {\ttfamily keeps\+Whitespace} option is set to {\ttfamily true}, leading and trailing whitespace are not trimmed, unless the {\ttfamily removes\+Indent} option is also enabled.
\item If the tag\textquotesingle{}s {\ttfamily removes\+Indent} option is set to {\ttfamily true}, any indentation that is shared by every line in the string is removed. This option is ignored unless {\ttfamily keeps\+Whitespace} is set to {\ttfamily true}.
\end{DoxyItemize}

{\bfseries{Note}}\+: If the tag text is the name of a symbol, and the symbol\textquotesingle{}s name includes leading or trailing whitespace (for example, the property names in `\{ ' \textquotesingle{}\+: true, \textquotesingle{} foo \textquotesingle{}\+: false \}\`{}), the tag text is not trimmed. Define a tag {\itshape modifier}, which is a function that, given a tag, will return a tag that is a subtag of the original. 

Applying the same modifier to a twice tag will return the same value ({\ttfamily m1(t1) == m1(t1)}) and applying multiple modifiers will, regardless or order, produce the same tag ({\ttfamily m1(m2(t1)) == m2(m1(t1))}).

When multiple modifiers are applied to a given base tag, each smaller set of modifiers is registered as a parent, so that for example {\ttfamily m1(m2(m3(t1)))} is a subtype of {\ttfamily m1(m2(t1))}, {\ttfamily m1(m3(t1)}, and so on.
