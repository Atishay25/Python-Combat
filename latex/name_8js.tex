Convert an array of doclet longnames into a tree structure, optionally attaching doclets to the tree. 

Each level of the tree is an object with the following properties\+:


\begin{DoxyItemize}
\item {\ttfamily longname \{string\}}\+: The longname.
\item {\ttfamily memberof \{string?\}}\+: The memberof.
\item {\ttfamily scope \{string?\}}\+: The longname\textquotesingle{}s scope, represented as a punctuation mark (for example, {\ttfamily \#} for instance and {\ttfamily .} for static).
\item {\ttfamily name \{string\}}\+: The short name.
\item {\ttfamily doclet \{Object?\}}\+: The doclet associated with the longname, or {\ttfamily null} if the doclet was not provided.
\item {\ttfamily children \{Object?\}}\+: The children of the current longname. Not present if there are no children.
\end{DoxyItemize}
