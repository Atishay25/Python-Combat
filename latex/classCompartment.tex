\doxysubsection{Detailed Description}
Extension compartments can be used to make a configuration dynamic. 

By \href{https://codemirror.net/6/docs/ref/\#state.Compartment.of}{\texttt{ wrapping}} part of your configuration in a compartment, you can later \href{https://codemirror.net/6/docs/ref/\#state.Compartment.reconfigure}{\texttt{ replace}} that part through a transaction.
