Create a highlighter style that associates the given styles to the given tags. 

The specs must be objects that hold a style tag or array of tags in their {\ttfamily tag} property, and either a single {\ttfamily class} property providing a static C\+SS class (for highlighter that rely on external styling), or a \href{https://github.com/marijnh/style-mod\#documentation}{\texttt{ {\ttfamily style-\/mod}}}-\/style set of C\+SS properties (which define the styling for those tags).

The C\+SS rules created for a highlighter will be emitted in the order of the spec\textquotesingle{}s properties. That means that for elements that have multiple tags associated with them, styles defined further down in the list will have a higher C\+SS precedence than styles defined earlier.
