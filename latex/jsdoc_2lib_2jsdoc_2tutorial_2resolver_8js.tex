Helper function that adds tutorial configuration to the {\ttfamily conf} variable. 

This helps when multiple tutorial configurations are specified in one object, or when a tutorial\textquotesingle{}s children are specified as tutorial configurations as opposed to an array of tutorial names.

Recurses as necessary to ensure all tutorials are added.
