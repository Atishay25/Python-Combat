Represents a document. 

A sample document is structured as follows\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<?\textcolor{keyword}{xml} \textcolor{keyword}{version}=\textcolor{stringliteral}{"1.0"} \textcolor{keyword}{encoding}=\textcolor{stringliteral}{"UTF-\/8"}?>}
\DoxyCodeLine{<\textcolor{keywordtype}{DOCTYPE} \textcolor{keyword}{html} \textcolor{keyword}{PUBLIC} \textcolor{stringliteral}{"-\///W3C//DTD XHTML 1.0 Strict//EN"}}
\DoxyCodeLine{                     \textcolor{stringliteral}{"http://www.w3.org/TR/xhtml1/DTD/xhtml1-\/strict.dtd"}>}
\DoxyCodeLine{<\textcolor{keywordtype}{html}>}
\DoxyCodeLine{    <\textcolor{keywordtype}{head}>}
\DoxyCodeLine{        <\textcolor{keywordtype}{title}>\textcolor{keyword}{My} \textcolor{keyword}{page} \textcolor{keyword}{title}</\textcolor{keywordtype}{title}>}
\DoxyCodeLine{    </\textcolor{keywordtype}{head}>}
\DoxyCodeLine{    <\textcolor{keywordtype}{body}>}

\DoxyCodeLine{        <\textcolor{keywordtype}{img} \textcolor{keyword}{src}=\textcolor{stringliteral}{"picture.png"}/>}
\DoxyCodeLine{    </\textcolor{keywordtype}{body}>}
\DoxyCodeLine{</\textcolor{keywordtype}{html}>}
\end{DoxyCode}


Each component of the document, such as the declaration, document type definition, and root element, are children of this node.

Documents must have exactly one element, which is the document\textquotesingle{}s root element.

Documents can have exactly one declaration and one document type definition in that order, so long as they precede the element.

Documents can have an unlimited number of comments or processing instructions, so long as they follow the declaration, if one exists. \mbox{\Hypertarget{XmlDocument_8js_a15e359ad5ad60be60cfeb208e5f3fee5}\label{XmlDocument_8js_a15e359ad5ad60be60cfeb208e5f3fee5}}
