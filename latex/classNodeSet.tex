\doxysubsection{Detailed Description}
A node set holds a collection of node types. 

It is used to compactly represent trees by storing their type ids, rather than a full pointer to the type object, in a numeric array. Each parser \href{\#lr.LRParser.nodeSet}{\texttt{ has}} a node set, and \href{\#common.TreeBuffer}{\texttt{ tree buffers}} can only store collections of nodes from the same set. A set can have a maximum of 2$\ast$$\ast$16 (65536) node types in it, so that the ids fit into 16-\/bit typed array slots.
