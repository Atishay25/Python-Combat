{\footnotesize\ttfamily var Xml\+Attribute}



Represents an attribute. 

An attribute is part of the start tag of an element and is structured as follows, where {\ttfamily \{name\}} is the name of the attribute and {\ttfamily \{value\}} is the value of the attribute\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<\textcolor{keywordtype}{element} \{\textcolor{keyword}{name}\}=\textcolor{stringliteral}{"\{value\}"}>}
\end{DoxyCode}


The {\ttfamily \{name\}} value is a property of this node, while the {\ttfamily \{value\}} property consists of the children of this node.

Attributes can have an unlimited number of attribute text, character references, and entity references. \mbox{\Hypertarget{XmlAttribute_8js_a2144db9aa6235a66571425badd855c94}\label{XmlAttribute_8js_a2144db9aa6235a66571425badd855c94}}
