\doxysubsection{Detailed Description}
\mbox{\hyperlink{classState}{State}} effects can be used to represent additional effects associated with a \href{https://codemirror.net/6/docs/ref/\#state.Transaction.effects}{\texttt{ transaction}}. 

They are often useful to model changes to custom \href{https://codemirror.net/6/docs/ref/\#state.StateField}{\texttt{ state fields}}, when those changes aren\textquotesingle{}t implicit in document or selection changes.
