\doxysubsection{Detailed Description}
The editor state class is a persistent (immutable) data structure. 

To update a state, you \href{https://codemirror.net/6/docs/ref/\#state.EditorState.update}{\texttt{ create}} a \href{https://codemirror.net/6/docs/ref/\#state.Transaction}{\texttt{ transaction}}, which produces a {\itshape new} state instance, without modifying the original object.

As such, {\itshape never} mutate properties of a state directly. That\textquotesingle{}ll just break things.
