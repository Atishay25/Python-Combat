Find the common prefix for an array of paths. 

If there is a common prefix, a trailing separator is appended to the prefix. Relative paths are resolved relative to the current working directory.

For example, assuming that the current working directory is {\ttfamily /\+Users/jsdoc}\+:


\begin{DoxyItemize}
\item For the single path {\ttfamily foo/bar/baz/qux.\+js}, the common prefix is {\ttfamily foo/bar/baz/}.
\item For paths {\ttfamily foo/bar/baz/qux.\+js}, {\ttfamily foo/bar/baz/quux.\+js}, and {\ttfamily foo/bar/baz.\+js}, the common prefix is {\ttfamily /\+Users/jsdoc/foo/bar/}.
\item For paths {\ttfamily ../jsdoc/foo/bar/baz/qux/quux/test.js}, {\ttfamily /\+Users/jsdoc/foo/bar/bazzy.js}, and {\ttfamily ../../\+Users/jsdoc/foo/bar/foobar.js}, the common prefix is {\ttfamily /\+Users/jsdoc/foo/bar/}.
\item For paths {\ttfamily foo/bar/baz/qux.\+js} and {\ttfamily ../../\+Library/foo/bar/baz.js}, there is no common prefix, and an empty string is returned.
\end{DoxyItemize}

Retrieve the fully qualified path to the requested resource. 

If the resource path is specified as a relative path, J\+S\+Doc searches for the resource in the following locations, in this order\+:


\begin{DoxyEnumerate}
\item The current working directory
\item The directory where the J\+S\+Doc configuration file is located
\item The J\+S\+Doc directory
\item Anyplace where {\ttfamily \mbox{\hyperlink{dist_2markdown-it_8js_a4b4eb847520ff4e45de16db801e26133}{require()}}} can find the resource (for example, in your project\textquotesingle{}s {\ttfamily node\+\_\+modules} directory)
\end{DoxyEnumerate}

If the resource path is specified as a fully qualified path, J\+S\+Doc searches for the resource in the following locations, in this order\+:


\begin{DoxyEnumerate}
\item The resource path
\item Anyplace where {\ttfamily \mbox{\hyperlink{dist_2markdown-it_8js_a4b4eb847520ff4e45de16db801e26133}{require()}}} can find the resource (for example, in your project\textquotesingle{}s {\ttfamily node\+\_\+modules} directory)
\end{DoxyEnumerate}
