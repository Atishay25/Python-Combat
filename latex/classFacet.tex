\doxysubsection{Detailed Description}
A facet is a labeled value that is associated with an editor state. 

It takes inputs from any number of extensions, and combines those into a single output value.

Examples of uses of facets are the \href{https://codemirror.net/6/docs/ref/\#state.EditorState^tabSize}{\texttt{ tab size}}, \href{https://codemirror.net/6/docs/ref/\#view.EditorView^editorAttributes}{\texttt{ editor attributes}}, and \href{https://codemirror.net/6/docs/ref/\#view.EditorView^updateListener}{\texttt{ update listeners}}.
