\doxysubsection{Detailed Description}
A change description is a variant of \href{https://codemirror.net/6/docs/ref/\#state.ChangeSet}{\texttt{ change set}} that doesn\textquotesingle{}t store the inserted text. 

As such, it can\textquotesingle{}t be applied, but is cheaper to store and manipulate.
