Represents an X\+ML element. 

A sample element is structured as follows, where {\ttfamily \{name\}} is the name of the element\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<\{\textcolor{keyword}{name}\} \textcolor{keyword}{attname}=\textcolor{stringliteral}{"attvalue"}>}
\DoxyCodeLine{    <\textcolor{keywordtype}{subelem}/>}
\DoxyCodeLine{    <?\textcolor{keyword}{pitarget} \textcolor{keyword}{picontent}?>}
\DoxyCodeLine{    \textcolor{keyword}{text}}
\DoxyCodeLine{</\{\textcolor{keyword}{name}\}></\textcolor{keywordtype}{pre}>}
\end{DoxyCode}


X\+ML elements can have an unlimited number of attributes, C\+D\+A\+TA sections, character references, comments, elements, entity references, processing instructions, and character data.

An element with no content will be represented using an empty element tag\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<\{\textcolor{keyword}{name}\}/>}
\end{DoxyCode}
