\doxysubsection{Detailed Description}
A change set represents a group of modifications to a document. 

It stores the document length, and can only be applied to documents with exactly that length.

\doxysubsubsection{\texorpdfstring{map()}{map()}}
{\footnotesize\ttfamily Change\+Set\+::map (\begin{DoxyParamCaption}\item[{}]{other,  }\item[{}]{before = {\ttfamily false} }\end{DoxyParamCaption})}



Given another change set starting in the same document, maps this change set over the other, producing a new change set that can be applied to the document produced by applying {\ttfamily other}. 

When {\ttfamily before} is {\ttfamily true}, order changes as if {\ttfamily this} comes before {\ttfamily other}, otherwise (the default) treat {\ttfamily other} as coming first.

Given two changes {\ttfamily A} and {\ttfamily B}, {\ttfamily A.\+compose(B.\+map(\+A))} and {\ttfamily B.\+compose(A.\+map(\+B, true))} will produce the same document. This provides a basic form of \href{https://en.wikipedia.org/wiki/Operational_transformation}{\texttt{ operational transformation}}, and can be used for collaborative editing. \mbox{\Hypertarget{classChangeSet_aef211f3569ba399256c8011287ec125b}\label{classChangeSet_aef211f3569ba399256c8011287ec125b}}
