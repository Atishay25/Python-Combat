{\footnotesize\ttfamily var Xml\+Char\+Ref}



Represents a character reference. 

A character reference is structured as follows, where {\ttfamily \{dec\}} is the decimal representation code point corresponding to a particular Unicode character\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\&\#\{\textcolor{keyword}{dec}\};}
\end{DoxyCode}


The corresponding hexadecimal version is structured as follows, where {\ttfamily \{hex\}} is the hexadecimal representation code point corresponding to a particular Unicode character\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\&\#\textcolor{keyword}{x}\{\textcolor{keyword}{hex}\};}
\end{DoxyCode}


Unicode characters outside of the Basic Multilingual Plane are represented using a surrogate pair consisting of two character references.

The {\ttfamily \{dec\}} and {\ttfamily \{hex\}} values are defined by the {\ttfamily char} and {\ttfamily hex} properties of this node; the former is the character to be represented while the latter indicates whether the decimal or hexadecimal representation should be used.
