{\footnotesize\ttfamily function make\+Default\+Param\+Finisher (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [private]}}



For functions that may have at least one parameter with default values, create a function that will automatically add the parameters\textquotesingle{} default values to the function\textquotesingle{}s documentation. 

If any default value is already documented, the function\textquotesingle{}s doclet will remain unchanged.

This function is only intended to handle default parameters whose node type is {\ttfamily Syntax.\+Literal} (string, numeric, and boolean literals). This is because more complex default values may include, for example, references to internal variables, which it may not make sense to include in documentation.
