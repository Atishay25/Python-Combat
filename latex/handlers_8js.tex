Create a doclet for a {\ttfamily symbol\+Found} event. 

The doclet represents an actual symbol that is defined in the code.

Here\textquotesingle{}s why this function is useful. A J\+S\+Doc comment can define a symbol name by including\+:


\begin{DoxyItemize}
\item A {\ttfamily @name} tag
\item Another tag that accepts a name, such as {\ttfamily @function}
\end{DoxyItemize}

When the J\+S\+Doc comment defines a symbol name, we treat it as a \char`\"{}virtual comment\char`\"{} for a symbol that isn\textquotesingle{}t actually present in the code. And if a virtual comment is attached to a symbol, it\textquotesingle{}s possible that the comment and symbol have nothing to do with one another.

To handle this case, this function checks the new doclet to see if we\textquotesingle{}ve already added a name property by parsing the J\+S\+Doc comment. If so, this method creates a replacement doclet that ignores the attached J\+S\+Doc comment and only looks at the code. \mbox{\Hypertarget{handlers_8js_ac44473e158366bf6037de322b3c345f3}\label{handlers_8js_ac44473e158366bf6037de322b3c345f3}}
