\doxysubsection{Detailed Description}
Information about a Java\+Script package. 

J\+S\+Doc can extract package information from {\ttfamily package.\+json} files that follow the \href{https://www.npmjs.org/doc/files/package.json.html}{\texttt{ npm specification}}.

{\bfseries{Note}}\+: J\+S\+Doc does not validate or normalize the contents of {\ttfamily package.\+json} files. If your {\ttfamily package.\+json} file does not follow the npm specification, some properties of the {\ttfamily \mbox{\hyperlink{classPackage}{Package}}} object may not use the format documented here.
