\doxysubsection{Detailed Description}
A language object manages parsing and per-\/language \href{https://codemirror.net/6/docs/ref/\#state.EditorState.languageDataAt}{\texttt{ metadata}}. 

\mbox{\hyperlink{classParse}{Parse}} data is managed as a \href{https://lezer.codemirror.net}{\texttt{ Lezer}} tree. The class can be used directly, via the \href{https://codemirror.net/6/docs/ref/\#language.LRLanguage}{\texttt{ {\ttfamily L\+R\+Language}}} subclass for \href{https://lezer.codemirror.net/}{\texttt{ Lezer}} LR parsers, or via the \href{https://codemirror.net/6/docs/ref/\#language.StreamLanguage}{\texttt{ {\ttfamily Stream\+Language}}} subclass for stream parsers.
