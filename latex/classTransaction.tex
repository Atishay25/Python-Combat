\doxysubsection{Detailed Description}
Changes to the editor state are grouped into transactions. 

Typically, a user action creates a single transaction, which may contain any number of document changes, may change the selection, or have other effects. Create a transaction by calling \href{https://codemirror.net/6/docs/ref/\#state.EditorState.update}{\texttt{ {\ttfamily Editor\+State.\+update}}}, or immediately dispatch one by calling \href{https://codemirror.net/6/docs/ref/\#view.EditorView.dispatch}{\texttt{ {\ttfamily Editor\+View.\+dispatch}}}.
