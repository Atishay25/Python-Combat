\doxysubsection{Detailed Description}
Helper class used to make it easier to maintain decorations on visible code that matches a given regular expression. 

To be used in a \href{https://codemirror.net/6/docs/ref/\#view.ViewPlugin}{\texttt{ view plugin}}. Instances of this object represent a matching configuration.
