Create a text cursor. 

The query is the search string, {\ttfamily from} to {\ttfamily to} provides the region to search.

When {\ttfamily normalize} is given, it will be called, on both the query string and the content it is matched against, before comparing. You can, for example, create a case-\/insensitive search by passing {\ttfamily s =$>$ s.\+to\+Lower\+Case()}.

\mbox{\hyperlink{classText}{Text}} is always normalized with \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/normalize}{\texttt{ {\ttfamily .normalize(\char`\"{}\+N\+F\+K\+D\char`\"{})}}} (when supported).
