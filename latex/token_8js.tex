new \mbox{\hyperlink{classToken}{Token(type, tag, nesting)}}

Create new token and fill passed properties. Token\+::type -\/$>$ String

Type of the token (string, e.\+g. \char`\"{}paragraph\+\_\+open\char`\"{})

Token\+::tag -\/$>$ String

html tag name, e.\+g. \char`\"{}p\char`\"{}

Token\+::attrs -\/$>$ Array

Html attributes. Format\+: {\ttfamily \mbox{[} \mbox{[} name1, value1 \mbox{]}, \mbox{[} name2, value2 \mbox{]} \mbox{]}}

Token\+::map -\/$>$ Array

Source map info. Format\+: {\ttfamily \mbox{[} line\+\_\+begin, line\+\_\+end \mbox{]}}

Token\+::nesting -\/$>$ Number

Level change (number in \{-\/1, 0, 1\} set), where\+:


\begin{DoxyItemize}
\item {\ttfamily 1} means the tag is opening
\item {\ttfamily 0} means the tag is self-\/closing
\item {\ttfamily -\/1} means the tag is closing
\end{DoxyItemize}

Token\+::level -\/$>$ Number

nesting level, the same as {\ttfamily state.\+level}

Token\+::children -\/$>$ Array

An array of child nodes (inline and img tokens)

Token\+::content -\/$>$ String

In a case of self-\/closing tag (code, html, fence, etc.), it has contents of this tag.

Token\+::markup -\/$>$ String

\textquotesingle{}$\ast$\textquotesingle{} or \textquotesingle{}\+\_\+\textquotesingle{} for emphasis, fence string for fence, etc.

Token\+::info -\/$>$ String

Additional information\+:


\begin{DoxyItemize}
\item Info string for \char`\"{}fence\char`\"{} tokens
\item The value \char`\"{}auto\char`\"{} for autolink \char`\"{}link\+\_\+open\char`\"{} and \char`\"{}link\+\_\+close\char`\"{} tokens
\item The string value of the item marker for ordered-\/list \char`\"{}list\+\_\+item\+\_\+open\char`\"{} tokens
\end{DoxyItemize}

Token\+::meta -\/$>$ Object

A place for plugins to store an arbitrary data

Token\+::block -\/$>$ Boolean

True for block-\/level tokens, false for inline tokens. Used in renderer to calculate line breaks

Token\+::hidden -\/$>$ Boolean

If it\textquotesingle{}s true, ignore this element when rendering. Used for tight lists to hide paragraphs.
