Create a mark decoration, which influences the styling of the content in its range. 

Nested mark decorations will cause nested D\+OM elements to be created. Nesting order is determined by precedence of the \href{https://codemirror.net/6/docs/ref/\#view.EditorView^decorations}{\texttt{ facet}}, with the higher-\/precedence decorations creating the inner D\+OM nodes. Such elements are split on line boundaries and on the boundaries of lower-\/precedence decorations. \mbox{\Hypertarget{classDecoration_a25936eec4d3a22b971ceec0ab4a50177}\label{classDecoration_a25936eec4d3a22b971ceec0ab4a50177}}
