Detailed Description

A decoration provides information on how to draw or style a piece of content.

You'll usually use it wrapped in a Range, which adds a start and end position. @nonabstract

◆ mark()

static Decoration::mark (   spec)
static

Create a mark decoration, which influences the styling of the content in its range.

Nested mark decorations will cause nested DOM elements to be created. Nesting order is determined by precedence of the facet, with the higher-precedence decorations creating the inner DOM nodes. Such elements are split on line boundaries and on the boundaries of lower-precedence decorations.
