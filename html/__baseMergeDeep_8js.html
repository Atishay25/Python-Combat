A specialized version of baseMerge for arrays and objects which performs deep merges and tracks traversed objects enabling objects with circular references to be merged.

Parameters
{Object}object The destination object.
{Object}source The source object.
{string}key The key of the value to merge.
{number}srcIndex The index of source.
{Function}mergeFunc The function to merge values.
{Function}[customizer] The function to customize assigned values.
{Object}[stack] Tracks traversed source values and their merged counterparts.
