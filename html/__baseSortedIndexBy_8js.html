private

The base implementation of _.sortedIndexBy and _.sortedLastIndexBy which invokes iteratee for value and each element of array to compute their sort ranking.

The iteratee is invoked with one argument; (value).

Parameters
{Array}array The sorted array to inspect.
{*}value The value to evaluate.
{Function}iteratee The iteratee invoked per element.
{boolean}[retHighest] Specify returning the highest qualified index.
Returns
{number} Returns the index at which value should be inserted into array.
