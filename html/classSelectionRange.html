Detailed Description

A single selection range.

When allowMultipleSelections is enabled, a selection may hold multiple ranges. By default, selections hold exactly one range.

get SelectionRange::assoc ( )

If this is a cursor that is explicitly associated with the character on one of its sides, this returns the side.

-1 means the character before its position, 1 the character after, and 0 means no association.
