Detailed Description

Information about a JavaScript package.

JSDoc can extract package information from package.json files that follow the npm specification.

Note: JSDoc does not validate or normalize the contents of package.json files. If your package.json file does not follow the npm specification, some properties of the Package object may not use the format documented here.
