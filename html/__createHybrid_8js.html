Creates a function that wraps func to invoke it with optional this binding of thisArg, partial application, and currying.

Parameters
{Function|string}func The function or method name to wrap.
{number}bitmask The bitmask flags. See createWrap for more details.
{*}[thisArg] The this binding of func.
{Array}[partials] The arguments to prepend to those provided to the new function.
{Array}[holders] The partials placeholder indexes.
{Array}[partialsRight] The arguments to append to those provided to the new function.
{Array}[holdersRight] The partialsRight placeholder indexes.
{Array}[argPos] The argument positions of the new function.
{number}[ary] The arity cap of func.
{number}[arity] The arity of func.
Returns
{Function} Returns the new wrapped function.
