Detailed Description

A parse stack.

These are used internally by the parser to track parsing progress. They also provide some properties and methods that external code such as a tokenizer can use to get information about the parse state.

◆ canShift()

Stack::canShift (   term)

Check if the given term would be able to be shifted (optionally after some reductions) on this stack.

This can be useful for external tokenizers that want to make sure they only provide a given token when it applies.
