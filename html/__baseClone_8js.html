function baseClone (   value,
  bitmask,
  customizer,
  key,
  object,
  stack 
)
private

The base implementation of _.clone and _.cloneDeep which tracks traversed objects.

Parameters
{*}value The value to clone.
{boolean}bitmask The bitmask flags. 1 - Deep clone 2 - Flatten inherited properties 4 - Clone symbols
{Function}[customizer] The function to customize cloning.
{string}[key] The key of value.
{Object}[object] The parent object of value.
{Object}[stack] Tracks traversed objects and their clone counterparts.
Returns
{*} Returns the cloned value.
