Populate the given dictionary with the appropriate JSDoc tag definitions.

If the tagDefinitions parameter is omitted, JSDoc uses its configuration settings to decide which tags to add to the dictionary.

If the tagDefinitions parameter is included, JSDoc adds only the tag definitions from the tagDefinitions object. The configuration settings are ignored.

Parameters
{module:jsdoc/tag/dictionary}dictionary
{Object}[tagDefinitions] - A dictionary whose values define the rules for a JSDoc tag.
