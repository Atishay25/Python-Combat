Detailed Description

Objects of this type provide context information and helper methods to indentation functions registered on syntax nodes.

Member Function Documentation

◆ baseIndent()

get TreeIndentContext::baseIndent ( )

Get the indentation at the reference line for this.node, which is the line on which it starts, unless there is a node that is not a parent of this node covering the start of that line.

If so, the line at the start of that node is tried, again skipping on if it is covered by another such node.
