Parameters
treeThe inner tree.
overlayIf this is null, this tree replaces the entire node (it will be included in the regular iteration instead of its host node). If not, only the given ranges are considered to be covered by this tree. This is used for trees that are mixed in a way that isn't strictly hierarchical. Such mounted trees are only entered by [`resolveInner`](#common.Tree.resolveInner) and [`enter`](#common.SyntaxNode.enter).
