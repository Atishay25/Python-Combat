◆ IterMode

function IterMode
Initial value:
{
IterMode[IterMode["ExcludeBuffers"] = 1] = "ExcludeBuffers"

Options that control iteration.

This option only applies in enter-style methods.

By default, regular mounted nodes replace their base node in iteration.

Enable this to make iteration include anonymous nodes (such as the nodes that wrap repeated grammar constructs into a balanced tree).

Can be combined with the | operator to enable multiple ones.

Enable this to ignore them instead.

It tells the library to not enter mounted overlays if one covers the given position.
