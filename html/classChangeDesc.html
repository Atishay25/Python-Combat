Detailed Description

A change description is a variant of change set that doesn't store the inserted text.

As such, it can't be applied, but is cheaper to store and manipulate.

ChangeDesc::iterChangedRanges (   f,
  individual = false

Iterate over the ranges changed by these changes.

(See ChangeSet.iterChanges for a variant that also provides you with the inserted text.) fromA/toA provides the extent of the change in the starting document, fromB/toB the extent of the replacement in the changed document.

When individual is true, adjacent changes (which are kept separate for position mapping) are reported separately.
