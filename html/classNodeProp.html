Detailed Description

Each node type or individual tree can have metadata associated with it in props.

Instances of this class represent prop names.

NodeProp::add (   match)

This is meant to be used with NodeSet.extend or LRParser.configure to compute prop values for each node type in the set.

Takes a match object or function that returns undefined if the node type doesn't get this prop, and the prop's value if it does.
